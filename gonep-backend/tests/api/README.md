# API Tests

This folder contains API endpoint test scripts.

## Files

- `test-api.js` - General API testing
- `test-auth.js` - Authentication API testing
- `test-demo-apis.js` - Demo API testing
- `test-demo-system.js` - Demo system testing
- `test-upload.js` - File upload testing
- `test-video-route.js` - Video route testing
- `test-video-upload.js` - Video upload testing

## Usage

Run these tests to verify your API endpoints are working correctly after deployment.
