# Database Tests

This folder contains database-related test and utility scripts.

## Files

- `check-db.js` - Check database connection
- `check-db-status.js` - Check database status
- `check-users.js` - Check user accounts
- `create-admin.js` - Create admin user
- `create-blog-tables.js` - Create blog tables manually
- `create-missing-tables.js` - Create missing tables

## Usage

These are utility scripts for database management and testing. Most functionality has been consolidated into the main migration script in the `migrations/` folder.
